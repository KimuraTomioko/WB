Добрый день. Исполняемый файл - main.py. Все файлы сохраняются в папку outputs_folder после работы парсера. Файл input.xlsx нужно закидывать в главную директорию. 

Вы можете поставить парсер на сервер следуя шагам:

Для Linux:
1. Купить и оплатить сервер.
2. Зайти на сервер при помощи программы Putty.
3. Скачать все зависимости на сервер(openpyxl, selenium).
4. Скачать git.
5. Скачать tmux. 
6. Скопировать репозиторий с кодом командый git clone https://github.com/KimuraTomioko/WB.git
7. Выполните команду: tmux
8. Перейдите в директорию с проектом, и запустите main.py


Windows Server 2016:

1. Зайти на сервер и перейти на сайт установки Python(https://www.python.org/)
2. Скачать Python оттуда
3. Открыть CMD и вписать команды(
    pip install selenium
    pip install openpyxl
)
4. Перекиньте проект на сервер(Если нет такой возможности, то скачайте git(https://git-scm.com/download/win) и скопируйте репозиторий на рабочий стол командой git clone https://github.com/KimuraTomioko/WB.git)
5. Откройте папку с программой и вызовите там CMD
6. Введите команду python main.py

Если вы сделали всё правильно, то всё будет работать.
